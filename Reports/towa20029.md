# 課題１:
##  Prototypeパターン

### 概要
* インスタンスを実装する時のデザインパターン。クラスからインスタンスを生成するのではなく、インスタンスから別のインスタンスをコピーして生成する。

* 概略図
```mermaid
classDiagram
    class Prototype{
      <<interface>>
      createClone()
    }
    class ConcretePrototype{
      createClone()
    }
    class Client{

    }
    Prototype -- Client:USes▶
    Prototype <|-- ConcretePrototype
```
